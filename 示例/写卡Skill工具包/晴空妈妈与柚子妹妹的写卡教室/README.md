# 晴空妈妈与柚子妹妹的写卡教室

## 一份从零开始的角色卡手写教程

---

**晴空妈妈** ☀️ 温柔耐心的教程讲解者。她会用最细致的方式把每一个概念揉碎了喂给你。

**柚子妹妹** 🍊 调皮好奇的新手作者。她会犯所有新手会犯的错，问所有新手想问的问题。

---

## 示例角色

整套教程以 **陈乖乖** 为贯穿示例。她是一个曾经被写成七个极端标签（"极度自卑""极致善良""被动顺从"……）的AI直出角色。在教程中，晴空妈妈带着柚子妹妹一步步把她从标签堆里救出来，重新写成了一个有调色盘、有三面性、有混色画面、有核心人格层、有意象的完整的人。

---

## 阅读建议

- 如果你完全不知道怎么下手，从第一章开始顺序读。
- 如果你已经会写基础信息和调色盘，可以直接跳到第三章三面性。
- 混色、核心人格层、意象都是进阶选项，你的角色需要时再用。
- 每一章结尾都有"小柚子的课后笔记"，可以当速查用。

---

## 写卡流程速查

```
第一步：基础信息 → 这个人是谁
第二步：调色盘   → 这个人是什么质地
第三步：三面性   → 不同场景怎么分流（如果需要）
第四步：混色画面 → 颜色怎么在同一瞬间里撞（如果适合）
第五步：核心人格层 → 这个人为什么做这种选择（如果需要）
第六步：二次解释 → 拦住AI的误读
第七步：意象     → 用一个画面收束一切
