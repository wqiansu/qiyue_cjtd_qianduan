# 📱 柚月小手机 (Yuzuki's Little Phone) / gaigai-phone

一个为 SillyTavern（酒馆）量身定制的**超拟真、高交互虚拟手机系统**。
打破传统文字 RP 的枯燥，让你的角色拥有真实的现代社交生活！通过微信聊天、刷朋友圈、看微博热搜、甚至是拨打视频通话，带来前所未有的沉浸式扮演体验。

---

## ✨ 核心特性 (Core Features)

### 🧠 革命性的大模型双轨通信

- **📴 线下模式 (剧情注入)**：智能抓取手机数据，在酒馆主聊天框中无缝切入角色的手机动态。
- **📱 线上模式 (独立静默)**：内置 `ApiManager`，支持手机内部独立连接 API（支持 OpenAI, Claude, Gemini, DeepSeek, Local 等）。在手机内聊天发微信、打视频，**完全不污染酒馆主线记录**！

### 🕹️ 原生物理级 UI 与交互

- **完美复刻手势**：全面接管浏览器手势，支持 iOS/微信 级别的**边缘右滑返回**。
- **Z-index 物理视图栈**：告别简陋的 `display: none`，采用真正的双层图层堆叠，滑动返回绝不白屏，极致丝滑。
- **全局时间引擎**：自动推算跨天、跨月、星期几，让手机时间与剧情时间完美同步。

### 🖼️ 告别 Base64 内存黑洞

- 内置 `ImageCropper` 缩放裁剪，全面劫持图片上传逻辑。所有头像、朋友圈配图、聊天发图直接上传至酒馆服务器本地 `/backgrounds`，只存路径，**彻底解决酒馆存档动辄几十MB导致卡死的问题**。

## 📲 内置丰富的应用生态 (App Ecosystem)

- 💬 **微信 (WeChat)**
  - 支持单聊/群聊、AI 智能生成联系人。
  - **特殊交互**：发送图片、表情包、转账、发红包（自带独立钱包资产评估体系）。
  - **音视频通话**：拟真来电弹窗，支持接入 TTS（MiniMax/OpenAI），AI 自动接听并语音连播！
  - **朋友圈**：角色自动发动态，支持点赞、楼中楼评论内嵌互动。
- 👁️‍🗨️ **微博 (Weibo)**
  - 根据剧情实时生成热搜榜单。
  - 高仿微博信息流，带有网感极强的路人评论与 IP 属地，支持一键转发到微信。
- 💕 **蜜语 (Honey)**
  - 专属直播互动平台，实时滚动弹幕、打赏礼物特效，支持视频背景上传、预留接入 NovelAI 画面提示词占位。
- 🎵 **音乐 (Music)**
  - 全局毛玻璃悬浮播放器，AI 可在剧情中推荐真实存在的歌曲并直接播放。
- 📔 **日记 (Diary)**
  - 根据聊天记录，AI 自动以角色第一人称视角撰写排版精美的实体书日记。
- 📞 **通话 (Phone)**
  - 记录所有接通/未接来电，随时回看通话语音文字转换 (Transcript)。

## 💡 安装与使用 (Installation)

*(请根据实际发布的获取方式填写)*

1. 将本项目文件夹 `gaigai-phone` 放入 SillyTavern 的 `public/plugins/` 目录下。
2. 重启 SillyTavern。
3. 在顶部导航栏或扩展菜单中找到“📱 柚月小手机”图标，点击即可唤出。
4. （可选）在设置中配置 TTS 密钥和独立 API 密钥，解锁最佳体验。
